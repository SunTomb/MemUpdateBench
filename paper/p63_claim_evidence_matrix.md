| --- | --- | --- | --- |
| Repeated same-slot updates expose a tradeoff among recoverability, stale burden, compactness, and answer robustness. | At k=16, methods differ sharply across state accuracy, EM/F1, stale burden, and memory size. | `paper/p63_metric_ledger.md`, `paper/p63_update_frequency_tradeoff.pdf` | Do not frame as a single-method leaderboard. |

| Heuristic CRUD also preserves final-state recoverability under exact slot lookup. | Heuristic CRUD slot-direct state accuracy is 1.00 for k=1,2,4,8,16. | `results/update_frequency_p63_summary/update_frequency_tables.md` | It still retains stale same-slot entries. |
| Stale same-slot burden causes answer collapse under slot-conditioned answering. | Raw append stale burden rises to 14.25 and slot-prompt EM/F1 drops to 0.07 / 0.10 at k=16. | `paper/p63_metric_ledger.md` | Phrase as strong evidence/diagnosis, not causal proof beyond this controlled setting. |
| Partial compaction is insufficient. | Heuristic CRUD has k=16 stale burden 7.44 and slot-prompt EM/F1 0.10 / 0.13. | `paper/p63_metric_ledger.md` | Do not imply all heuristic approaches must fail. |
| Long25 reduces stale burden and memory size. | Long25 k=16 stale burden is 1.13 and final memory size is 9.43. | `paper/p63_metric_ledger.md` | Do not present Long25 as a universal winner. |
| Long25 remains imperfectly reliable. | Long25 k=16 slot-direct state accuracy is 0.91. | `paper/p63_metric_ledger.md` | Attribute to missed final updates or hard distractors only where error analysis supports it. |
| Long25 k=16 failures are mainly wrong-value final-state errors. | Local analyzer finds 9 `wrong_value` state errors; remote rerun finds 8 `wrong_value` state errors, concentrated in `company` slots. | `paper/p63_error_analysis_k16.md`, `paper/remote_verification_log.md` | Do not generalize beyond the current hard split. |
| Constrained CRUD is an upper-bound diagnostic. | Constrained CRUD has slot-direct state accuracy 1.00 and stale burden 0.00 at k=16. | `paper/p63_metric_ledger.md` | Do not describe as a natural external-memory system. |
